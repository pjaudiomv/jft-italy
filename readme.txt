== Installation ==

This section describes how to install the plugin and get it working.

1. Download and install the plugin from WordPress dashboard. You can also upload the entire JFT Italy Plugin folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Add [jft_italy] shortcode to your Wordpress page/post or add widget JFT Italy to your sidebar, Footer etc.
